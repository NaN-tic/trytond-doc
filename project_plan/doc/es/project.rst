Planificación
=============

En un proyecto o tarea también podrá especificar fechas de ejecución. Esta
información la dispone en la pestaña *Planificación*.

* |leveling_delay|
* |back_leveling_delay|
* |early_start_time|
* |late_start_time|
* |early_finish_time|
* |late_finish_time|
* |actual_start_time|
* |actual_finish_time|
* |constraint_start_time|
* |constraint_finish_time|
